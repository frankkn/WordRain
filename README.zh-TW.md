# WordRain 🌧️⌨️

**單字像雨一樣落下。在它們碰到水面前打完 — 漏接太多,洪水就會淹沒整個畫面。**

🔗 立即遊玩:**https://word-rain-mu.vercel.app**

## 怎麼玩

打開網頁,選 **NEW GAME**,開始打字。

- 每顆雨滴帶著一個單字。打出字首鎖定它 — 多個單字符合時,鎖定**最接近水面**的那顆 — 接著把整個字打完,雨滴就會爆散消除。
- 打錯字只會中斷 combo,**不會重置進度**。你正在打的單字會放大顯示在水面上方,再混亂也不會跟丟目標。
- 每漏接一顆,水位就上升一格,而且雨勢會越來越快。水位到 40%,畫面淹沒、這局結束。
- 越長的字分數越高;連續消字不失誤,combo 加成越疊越高。

### 操作

| 按鍵 | 動作 |
|------|------|
| `↑` `↓` | 選單移動 |
| `←` `→` | 調整選項 · 切換排行榜難度 |
| `Enter` | 確認 · 再玩一局 |
| `Esc` | 暫停 · 返回 |
| `A`–`Z` | 打字! |

右上角的 🔊 按鈕,一鍵靜音整個遊戲。

## 難度與世界排行榜

**EASY / MEDIUM / HARD** 影響雨滴落速、生成頻率與單字長度。每個難度有**各自獨立的世界前 10 排行榜** — 只要一局打進前 10,就能留下名字和國家,讓全世界看到。主選單的 **LEADERBOARDS** 隨時可以瀏覽三張榜。

## 選項

**MUSIC** 音量、**SOUND** 音量、**DIFFICULTY** 難度、**THEME** 主題 — 方向鍵調整、即時生效,下次打開都還記得。你的個人最高分也會保存在本機。

### 字表主題

六種主題字表任你切換:**CLASSIC**(雨夜混搭)、**ANIMALS** 動物、**FOOD** 食物、**CODE** 程式術語(沒錯,`npm` 和 `refactor` 都算字)、**SPACE** 太空、**FANTASY** 奇幻。主題只改變單字內容 — 難度與排行榜不受影響。

## 技術

Vite + TypeScript,單一 Canvas 2D 渲染 — 沒有遊戲引擎、零 runtime 依賴。音效用 Web Audio 即時合成。世界排行榜由 Supabase 支撐。
