# WordRain 🌧️⌨️

A typing game where words fall from the sky like rain. Each raindrop carries a character — type it correctly before it hits the ground. Clear them fast, or the water rises and floods the screen. The rain falls faster over time.

## Gameplay

- Characters fall as raindrops from the top of the screen.
- Type the matching character to pop a drop before it lands.
- Missed drops raise the water level.
- Survive as long as you can — rain speed increases over time.
